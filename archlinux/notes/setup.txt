how the FDE was setup
=====================

create a keyfile
----------------
dd if=/dev/urandom bs=1024 count=8 | openssl aes256 -out root.key

figure out best cipher
----------------------
cryptsetup benchmark

LOL, it was serpent-xts-plain on my netbook, which makes no sense since it is faster than AES.

format
------
openssl aes256 -d -in root.key 2>/dev/null | cryptsetup --key-file=- -c serpent-xts-plain -s 512 -h whirlpool luksFormat /dev/sda1

mount
-----
openssl aes256 -d -in root.key 2>/dev/null | cryptsetup --key-file=- luksOpen luksOpen /dev/sda1 root

kernel
------
cp ssldec inicpio hooks and install from git repo to install
cp mkinitcpio.conf to /etc from git repo
