Apply the desired seabird theme to alacritty by pasting the corresponding code block into `alacritty.yml`.

### seagull

```
# Colors (seagull)
colors:
  # Default colors
  primary:
    background: '0xffffff'
    foreground: '0x6d767d'

  # Colors the cursor will use if `custom_cursor_colors` is true
  cursor:
    text: '0xffffff'
    cursor: '0x6d767d'

  # Normal colors
  normal:
    black:   '0x1d252b'
    red:     '0xff4053'
    green:   '0x11ab00'
    yellow:  '0xbf8c00'
    blue:    '0x0099ff'
    magenta: '0xff549b'
    cyan:    '0x00a5ab'
    white:   '0xe6eaed'

  # Bright colors
  bright:
    black:   '0x0b141a'
    red:     '0xff6200'
    green:   '0x61707a'
    yellow:  '0x6d767d'
    blue:    '0x787e82'
    magenta: '0x9854ff'
    cyan:    '0x808487'
    white:   '0xffffff'
```
